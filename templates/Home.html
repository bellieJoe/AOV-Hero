FLOWCHARTS

This figure shows the flow of the game. At first start of the game the narration of the main story of the game will appear and after that the game will bring you in the main menu of the game where the users will see the user interface of the game, the settings, shop, every level of the game, the weaponry etc. In main menu you can select different categories you like to choose, and also in the main menu you can start playing the game and be able to slay the monsters or the enemies on the game. In the main menu you can also shop different weapons and upgrade it to its max level, you can also upgrade your main character in the game in order to perform well during the game.
